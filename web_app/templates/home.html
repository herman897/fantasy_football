Welcome Home

Welcome! This is the Fantasy Football Advisor app home page. Please see below for instructions on running Fantasy Football Advisor's Features!

FF Report Tool

Fantasy Football Advisor's FF Report Tool can be run in the terminal (see instructions in the README file), The report will allow you to compare two players projections for a given week, or the entire season if you're preparing for your draft!

Team Page Tool

Navigate to to the Team Page in the top right and submit the team form, creating a customized page displaying your fantasy team! Please spell as accurately as possible since we use a fuzzy search to fetch your players. The team view will include statistics to help you set your lineup come gameday!
